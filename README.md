# openlawsvpn for Android

Android client for [openlawsvpn](https://github.com/openlawsvpn/openlawsvpn) — connect to AWS Client VPN endpoints with SAML/SSO.

## Build

```sh
make init   # initialize submodules
make deps   # cross-compile OpenSSL + LZ4
```

Open in Android Studio (AGP 9.x, NDK 30+).
